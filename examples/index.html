EntryStore.js samples

Note that before you can run any of the examples below you need to have an EntryStore instance running. Furthermore the webpages needs to be served from the same domain to avoid CORS problems. Hence, you should run the following commands:
  • yarn entrystore:examples
  • yarn examples
